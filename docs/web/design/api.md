## Drag&Drop API
: 사용자 편의성을 고려한 UI 기능
<br> 마우스를 사용하여 애플리케이션 간 파일이나 데이터를 전달하는 기능으로서 다양한 이벤트 핸들러를 구현하여 처리

- dragstart
- dragend
- dragenter
- dragover
- drop
