# 연산자 Operator

### 논리연산자 AND, OR, NOT

![1](../images/operator_1.png)

- AND : 데이터 2개가 모두 True인 경우에 True 반환
- OR : 데이터가 1개라도 True이면 True 반환
- NOT : True면 False 반환, False면 True 반환
``` python
a = 0 # False
b = 1 # True

print(bool(a and b)) # False
print(bool(a or b)) # True
print(not a) # True
```
